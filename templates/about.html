What is Etymograph?

For anyone interested in the origin of words, Etymograph is a website and API that shows etymological data in a structured and easy-to-use way.

The word database is open to change from the hands of users like you. Words and relationships can be added, modified, and flagged.

Etymograph's API allows researchers, students, or anyone inclined to quickly access the database from a command line interface or script.

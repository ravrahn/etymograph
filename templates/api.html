Etymograph API

The /descs query

Usage:

<API address>/<word_id>/descs

Returns a JSON object containing the etymological descendant of the word with the given ID. Etymological descendants include the descendants of descendants, forming a tree-like structure. This query can also take an additional 'depth' argument as a get or post parameter specifying how many layers of descendants to return (i.e. the maximum depth of the descendants tree).

Inputs:

word_id:Integer The ID of the word to get the descendents of
depth:Integer (Optional) The depth of descendants to return. NOTE: Entered as a GET/POST parameter of the form "depth=n"
